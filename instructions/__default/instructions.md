 # Welcome to the Apache Web Server Lab

In this lab, you will explore a basic Apache web server environment running inside an Instruqt container.

## Available Tabs

* **Server**
  Access the Apache web application running on the container.

* **Shell**
  Access the terminal to run Linux and Apache-related commands.

## Example Commands

Check Apache version:

```bash
httpd -v
```

Check running processes:

```bash
ps aux
```

## Goal

This lab demonstrates a simple HCL-based container setup with networking, service exposure, and terminal access using the Instruqt Labs environment.
